#Getting and Cleaning Data - Code Book 


## Description
The goal of this project is to show how to change raw data into a clean data set.

## Source Data
This data was obtained from the [ The UCI Machine Learning Repositoru]
  (https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip)
